# Tic Tac Toe Demo

This is a web-sockets with socket.io two player game of Tic Tac Toe  with socket.io.


## To test locally:
In VS Code open the terminal, make sure you are in the root directory (you can run the command pwd to double check).

Run `npm install`

Run `node app.js` to start the server


It uses the npm open module to open the server in your browser automatically.
